# Personal Portfolio Website

## Project Overview
A fully responsive personal portfolio website built for internship evaluation using HTML5, CSS3, Bootstrap 5, and JavaScript.

## Features
- Responsive navbar with smooth scrolling
- Hero section with typing animation
- About, Skills, Projects, Education, Contact sections
- Contact form with validation
- Dark/Light mode toggle
- Scroll-to-top button
- AOS animations
- Animated skill progress bars

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- AOS Library
- Font Awesome & Bootstrap Icons

## How to Run
1. Clone the repository
2. Open `index.html` in browser
3. Or use Live Server in VS Code
